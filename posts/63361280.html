hexoのbilibili追番页面

2021-03-29

Demo: bilibili番剧，
所使用到的插件：hexo-bilibili-bangumi

    [blogrootfolder]，为博客根目录，以下统一写成这种。

安装步骤

    安装插件hexo-bilibili-bangumi：

npm install hexo-bilibili-bangumi --save

在等待安装的同时，我们可以再开启一个git bash here，输入以下指令，新建一个页面：

hexo new page "bangumi"

注意：无需修改[blogrootfolder]/source/bangumi/index.md。他的配置完全取决于下面的设置。

在[blogrootfolder]下的_config.yml（非主题配置文件）末尾加入以下配置：

bangumi:
  enable: true
  path:
  vmid:
  title: '追番列表'
  quote: '生命不息，追番不止！'
  show: 1
  loading:
  metaColor:
  color:
  webp:
  progress:

其中，

  • enable: 是否启用。
  • path: 番剧页面路径，默认bangumi/index.html。
  • vmid: 哔哩哔哩番剧页面的vmid(uid)。
  • title: 该页面的标题。
  • quote: 写在页面开头的一段话，支持html语法，可留空。
  • show: 初始显示页面：0: 想看, 1: 在看, 2: 看过，默认为1。
  • loading: 图片加载完成前的loading图片
  • metaColor: meta部分(简介上方)字体颜色
  • color: 简介字体颜色
  • webp: 番剧封面使用webp格式(此格式在safari浏览器下不显示，但是图片大小可以缩小100倍左右), 默认true
  • progress: 获取番剧数据时是否显示进度条，默认true

    如何获取bilibiliuid？
  确保浏览器的bilibili在登录状态下，访问https://space.bilibili.com，此时，地址栏的最后一串数字即为uid。

部署

    部署过程也比较简单，在hexo g之后、hexo d之前，加入hexo bangumi -u即可。
    第一次使用hexo bangumi -u，往后部署无需再用，本地查看hexo server同理（大概吧）。

hexo cl && hexo g && hexo bangumi -u && hexo d
