python 配置

    先下载

Python！点击图中的Downloads。

    选择最新版本3.9.2。

    以下就是py安装程序辣！安装的时候，注意要选择最后一个Add Python 3.9 to Path。然后单击Install Now。

    默默等待其安装完成。

    在其安装完成之后，打开bash或cmd，输入python -V来查看py版本，出现版本号，则说明安装成功！

接下来：

  1. 在cmd输入pip list检查是否已存在需要的包；
  2. 输入pip install flake8下载安装flake8；
  3. 输入pip install yapf下载安装yapf。

安装 Vscode

：Vscode 官网

    安装过程不多说qwq！

    接下来，新建一个文件夹，名称随意，我就起了个py，然后用vscode打开。

    之后，先点击vscode左边的方框，再在旁边的文本框中输入python，然后下载第一个python，（先选中，然后点击安装）。

    点击第一个图标（如下图所示），新建一个test.py，随便写一点东西：

print("hello world!")

    就像这样↓，然后点击第二个，新建一个.vscode文件夹，

    如果出现了这个，点击安装即可，然后再等待其终端默默安装完成。

    此时，我们不是已经写好了一个test.py了嘛，确保其在打开状态而且你当前的页面就是test.py，然后点击F5运行代码：

    看到hello world了嘛！

可是没有黑漆漆的控制台，没那味(#`O′)

调试控制台

    在你的.vscode文件夹下，新建两个文件，分别是settings.json、launch.json，具体内容如下：

    launch.json:

{
  "configurations": [{
    "name": "Python",
    "type": "python",
    "request": "launch",
    "program": "${file}",
    "console": "externalTerminal",
    "stopOnEntry": false
  }]
}

    settings.json

{
    "python.linting.flake8Enabled": true,
    "python.formatting.provider": "yapf",
    "python.linting.flake8Args": ["--max-line-length=248"],
    "python.linting.pylintEnabled": false
}

    全部保存!
