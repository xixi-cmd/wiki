twikoo设置邮件提醒

2021-01-08

我用的评论系统：twikoo，评论邮箱：frank.ka@foxmail.com。
故，以下皆以foxmail邮箱为例。

# 邮件通知
SENDER_EMAIL: name@example.com    # 邮件通知邮箱地址。对于大多数邮箱服务商，SENDER_EMAIL 必须和 SMTP_USER 保持一致，否则无法发送邮件。
SENDER_NAME: 青葱博客评论提醒
SMTP_SERVICE:                     # 因为用的是foxmail邮箱，所以这里留空。如果是其他邮箱的，请按照设置说明配置。
SMTP_HOST: smtp.qq.com            # 虽然用的是foxmail，但它是QQ邮箱的英文邮箱，因此，需要自定义。
SMTP_PORT: 465                    # 自定义 SMTP 端口。如您已配置 SMTP_SERVICE，此项请留空。
SMTP_SECURE: true                 # 自定义 SMTP 是否使用TLS，请填写 true 或 false。如您已配置 SMTP_SERVICE，此项请留空。不懂，开就是了。
SMTP_USER: name@example.com       # 必须和SENDER_EMAIL一致。
SMTP_PASS:                        # 邮件通知邮箱密码，QQ邮箱请填写授权码。foxmail邮箱，在QQ邮箱网页端更新授权码。
MAIL_SUBJECT:                     # 自定义通知邮件主题，留空则使用默认主题。
MAIL_TEMPLATE:                    # 自定义通知邮件模板，留空则使用默认模板。可包含的字段：${SITE_URL}, ${SITE_NAME}, ${PARENT_NICK}, ${PARENT_COMMENT}, ${NICK}, ${COMMENT}, ${POST_URL}
MAIL_SUBJECT_ADMIN:               # 自定义博主通知邮件主题，留空则使用默认主题。
MAIL_TEMPLATE_ADMIN:              # 自定义博主通知邮件模板。留空则使用默认模板。可包含的字段：${SITE_URL}, ${SITE_NAME}, ${NICK}, ${COMMENT}, ${POST_URL}
