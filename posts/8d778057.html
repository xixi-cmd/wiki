Quick Start

Create a new post

hexo new "My New Post"

    More info: Writing

Run server

hexo server

More info: Server

Generate static files

hexo generate

More info: Generating

Deploy to remote sites

hexo deploy

More info: Deployment

快速开始

创建一篇文章

建议：先提前开启博客根目录的配置文件_config.yml中的一项配置：post_asset_folder: false，将其调为true。这样，创建完一篇文章后，会在/source/_posts/内生成和文章同名的文件夹，这个文件夹可以用来存储图片、资源等等。

    生成文章

hexo new "your post name"

    例如：

hexo new "hexo-blog"

    此时，你已经写完了文章。现在，你有两种选择：1. 直接部署； 2. hexo s本地查看。

直接部署

    直接部署，可以通过如下指令:

hexo clean && hexo generate && hexo deploy

本地查看

hexo clean && hexo generate && hexo server

    更多内容请查看：Writing、Server、Generating、Deployment。

    好啦，到此结束啦！欢迎讨论哦！
